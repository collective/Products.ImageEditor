Introduction
============
Once installed this product adds a "Image Editor" link near the image widget. 
You can also manually enable an Image Editor button.

Features
========

* rotate
* flip
* blur
* compress
* contrast
* brightness
* sharpen
* add drop shadows
* crop
* resize
* sepia
* save as another content type.

Dependencies
============

* collective.js.jqueryui
