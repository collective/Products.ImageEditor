Introduction
============
Once installed this product adds a "Image Editor" tab for the image content type and News Item content type.

Features
========

* rotate
* flip
* blur
* compress
* contrast
* brightness
* sharpen
* add drop shadows
* crop
* resize
* sepia
* save as another content type.

Roadmap
=======
