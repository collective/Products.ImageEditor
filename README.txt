Introduction
============
Once installed this product adds a "Image Editor" link near the image widget.

Features
========

* rotate
* flip
* blur
* compress
* contrast
* brightness
* sharpen
* add drop shadows
* crop
* resize
* sepia
* save as another content type.

Roadmap / Proposal
==================

* Categorize actions (effects, transformations, colors)

============
